What is CCTFF?

CCTFF (Custom Chat Text Formatting Framework) is a small Roblox framework that centralizes all chat tag and text-formatting logic into a single configurable module.

Instead of hard-coding tags, colors, and prefixes inside multiple chat scripts, CCTFF lets you register tags and formatting rules in one place and apply them consistently on both server and client.

Core idea

At the center of CCTFF is a configuration table that describes every tag and text rule your experience supports. From there, helper APIs let you attach tags to players, build final chat messages, and inject formatting into Roblox chat services. 


return {
    Tags = {
        Owner = {
            Priority = 100,
            PrefixText = "[OWNER]",
            TextColor = Color3.fromRGB(255, 196, 61),
        },

        Dev = {
            Priority = 90,
            PrefixText = "[DEV]",
            TextColor = Color3.fromRGB(125, 249, 255),
        },
    },

    Defaults = {
        MaxEquippedTags = 3,
    },
}

This configuration-first approach means you add, remove, or edit tags without touching your chat pipeline, which keeps both Studio iterations and long-term maintenance clean.

High-level flow

The framework separates configuration, server APIs, and client rendering so that chat tags stay consistent regardless of where they’re displayed. 


local ReplicatedStorage = game:GetService("ReplicatedStorage")
local CCTFF = require(ReplicatedStorage:WaitForChild("CCTFF"))

local TagAPI = CCTFF.TagAPI

game.Players.PlayerAdded:Connect(function(player)
    -- Give and equip a tag
    TagAPI:Give(player, "Dev")
    TagAPI:Equip(player, "Dev")
end)
