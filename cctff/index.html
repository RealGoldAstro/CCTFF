Custom Chat Text Formatting Framework

CCTFF is a lightweight Roblox framework for building rich, configurable chat formatting and tags based on a single, central configuration instead of scattered magic values.

Quick install preview

ServerScriptService.SetupCCTFF 

-- Load the framework
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local CCTFF = require(ReplicatedStorage:WaitForChild("CCTFF"))

-- Configure your tags in one place
CCTFF:RegisterTag("Owner", {
    Priority = 100,
    PrefixText = "[OWNER]",
    TextColor = Color3.fromRGB(255, 196, 61),
})

-- Hook into your chat system and apply formatting
CCTFF:AttachToTextChatService()
